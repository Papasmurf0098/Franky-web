# Franky-web

Static dashboard for Franky Frontier.

## Run locally

### Prerequisites
- Node.js 18+ (tested with Node 22)

### Start
```bash
npm start
```

The app will be available at:

- `http://localhost:4173`

### Optional environment variables
- `PORT` (default: `4173`)
- `HOST` (default: `0.0.0.0`)
